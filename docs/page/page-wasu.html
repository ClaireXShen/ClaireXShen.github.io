WASU Kids

WASU Kids is a digital media ecosystem for children and adolescents, which allows children of all ages to enjoy media entertainment through accessible interaction, freeing parents to manage children's screen time. This project aims to provide a potential solution for multiple users sharing one account in the customer dimension.

Role: UX designer

Duration: 8 weeks

Goal:

  • Develop the minimum viable product (MVP) based on current media resources for obtaining feedback.
  • Take the expandability into account when drafting the framework and interaction rules.
  • The essential features of MVP are Parental Control, Role Switch, and Intuitional Interaction Flow.
  • The product would be TV-based and consider parents’ needs as a priority.
